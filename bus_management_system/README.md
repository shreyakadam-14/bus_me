# Bus Management System

A comprehensive bus management system built with PyQt5 and SQLite.

## Features

- User Authentication & Role-based Access
- Bus Management with Insurance Tracking
- Driver Management
- School Management
- Reports Dashboard
- System Administration
- Settings Management

## Requirements

- Python 3.7 or higher
- PyQt5
